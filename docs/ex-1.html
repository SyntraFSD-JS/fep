Oefening 1

Setup:

Install vue-cli


        npm install -g @vue/cli

Create a new project


        vue create fep
        # Select 'Manually select feature'
        # Pick (with space) 'Babel', 'Router' and 'Linter / Formatter'
        # You want to user history mode for router
        # Select 'ESLint + Prettier'
        # Select 'Lint on save'
        # Select 'In dedicated config files'

Run the development server


        cd fep
        npm run serve

Cleanup

Kuis het default project op:

  • Alle styling (css) mag weg.
  • Alle components mogen weg.

New views + component

  • Zorg ervoor dat er 5 pagina's zijn (views)
    • Home
    • Register
    • Login
    • Account
    • Board
  • Configureer het router.js bestand, zodat de 5 paginas gelinkt zijn
  • Maak een Header component waarin de 5 paginas gelinkt zijn (router-link)
  • Zet je Header component in App.vue

Over and out
